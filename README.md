## Détection de Fraude Bancaire

Projet data analyst - détection de transactions frauduleuses sur données bancaires réelles.

## Objectif

Identifier les fraudes parmi 284 807 transactions de cartes bancaires européennes (0,17%% de fraudes).

## Dataset

- Source : [Kaggle ULB](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)
- 284 807 transactions · 31 variables · 492 fraudes

## Stack technique

- Python · Pandas · Scikit-learn · Matplotlib · Seaborn
- Jupyter Notebook · Git / GitHub
